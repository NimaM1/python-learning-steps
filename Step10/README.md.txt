This program manages an online library using the OpenLibrary API and local JSON files.

Features:

Search for books by title and borrow them

Search for author information and their top books

View a list of borrowed books

Return borrowed books

Each search and borrowing action is saved locally for history tracking (onlineLibrary_searchHistory.json and borrow.json).

This project teaches working with APIs, JSON storage, user input handling, and basic library management operations.
