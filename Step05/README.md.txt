A beginner password manager that stores website login credentials in a JSON file.

It includes:

Add a new saved account (site name, username, password)

View all saved accounts (password partially hidden)

Search for a site and verify with a random security number before showing full password

Update username or password after verifying old values

Delete saved login information

Exit option

This project teaches JSON data storage, input validation, loops, search functionality, and basic security concepts (masking passwords, simple verification).
